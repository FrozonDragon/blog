Components of shell command

  • Options typically consist of a single letter preceded by a dash;
    • several options can be combined after a single dash;
    • a single option consists of a word, it is usually preceded by a double dash (–)
  • Arguments
  • Environment variables stores information that may be useful to the user’s shell session
  • Metacharacters are characters that have special meaning to the shell, e.g.
    • To direct the output of a command to a file (>)
    • To pipe the output to another command (|)
    • To run a command in the background (&)

Command-Line Editing

Keystrokes for Navigating Command Lines

Keystroke Meaning
Ctrl+F Go forward one character
Ctrl+B Go backward one character
Alt+F Go forward one word
Alt+B Go backward one word
Ctrl+A Go to the beginning of the current line
Ctrl+E Go to the end of the line
Ctrl+L Clear screen and leave line at the top of the screen

Keystrokes for Editing Command Lines

Keystroke Meaning
Ctrl+D Delete the current character
Backspace Delete the previous character
Ctrl+T Switch positions of current and previous characters
Alt+T Switch positions of current and previous characters
Alt+U Change the current word to uppercase
Alt+L Change the current word to lowercase
Alt+C Change the current word to an initial capital
Ctrl+V Add a special character. For example,to add a Tab character, press Ctrl+V+Tab

Keystrokes for Cutting and Pasting Text

Keystroke Meaning
Ctrl+K Cut text to the end of the line
Ctrl+U Cut text to the beginning of the line
Ctrl+W Cut the word located behind the cursor
Alt+D Cut the word following the cursor
Ctrl+Y Paste most recently cut text
Alt+Y Rotate back to previously cut text and paste it
Ctrl+C Delete the entire line

Command-Line Completion

  • Environment variable — shell completes the text with an environment variable from the current shell if it type begins with a dollar sign ($)
  • Username—If the text you type begins with a tilde (~), the shell completes the text with a username.
  • Command, alias, or function—If the text you type begins with regular characters, the shell tries to complete the text with a command, alias, or function name.
  • Host name—If the text you type begins with an at (@) sign, the shell completes the text with a host name taken from the /etc/hosts file.

Command-Line Recall

  • history command is to view your history list
  • history command followed by a number will list that many of the most recent commands
  • Recall one of those commands using an exclamation point !
  • fc command followed by a history line number will open in a text editor for you to edit the command, quit to execute

    Practice 6: Try Command-Line Recall

The history list is stored in the .bash_history file defined in $HISTFILE in your home directory. Up to 1,000 (defined in $HISTSIZE) history commands are stored for you by default.

  • Bash maintains the list of commands internally in memory while it’s running. They are written into .bash_history on exit.
  • When an interactive shell exits, the last $HISTSIZE lines are copied from the history list to the file named by $HISTFILE

Practice 7: Check history file

Some people disable the history feature for the root user by setting the HISTFILE to /dev/null or simply leaving HISTSIZE blank. This prevents information about the root user’s activities from potentially being exploited.

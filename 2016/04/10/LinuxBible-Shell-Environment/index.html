Reading Note for Linux Bible: Shell Enviroment

Bash Configuration Files

File for Whom When Executed What Info Remarks
/etc/profile every user log in path; location of your mailbox ; the size of your history files Gathers shell settings from configuration files in the /etc/profile.d directory.
/etc/bashrc every user each time a bash shell is opened sets the default prompt ; may add one or more aliases. Values in this file can be overridden by information in each user’s ~/.bashrc file.
~/.bash_profile each user user logs in and executed only once information that is specific to his or her own use of the shell By default it sets a few environment variables and executes the user’s .bashrc file.
~/.bashrc each user log in and also each time you open a new bash shell information that is specific to your bash shells. This is the best location to add environment variables and aliases so that your shell picks them up.
~/.bash_logout each user Executes each time you log out By default, it simply clears your screen.
